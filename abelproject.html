API anahtarınız sizin kontrolünüzde

Bu uygulama anahtarınızı yalnızca bu tarayıcının yerel depolama alanında saklar. Anahtarınız herhangi bir sunucuya, öğretmene veya başka bir kullanıcıya gönderilmez.

  • Gemini ile bağlantı, plan oluşturma isteği sırasında tarayıcınızdan doğrudan kurulur.
  • İstediğiniz zaman Ayarlar bölümünden anahtarınızı değiştirebilir veya Tüm Verileri Temizle seçeneğiyle silebilirsiniz.
  • Paylaşımlı bilgisayarlarda işiniz bittiğinde tüm verileri temizlemenizi öneririz.
API anahtarını adım adım nasıl alırım?
  1. Google hesabınızla Google AI Studio sayfasını açın.
  2. Create API key seçeneğine basın ve oluşturulan anahtarı kopyalayın.
  3. Anahtarı bu sayfaya yapıştırın; sistem geçerliliğini kontrol eder.
  4. Doğrulama tamamlanınca plan üretmeye başlayabilirsiniz.
